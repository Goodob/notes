#+TITLE: Struts2




* 框架
框架是一个半成品。它提供了一些 api 或者实用方法给我们直接调用。
所以使用框架，相当于在别人开发了一半的项目上继续开发。
因此，框架有以下优势：
1. 能提高我们的开发效率。
2. 能提高代码的健壮性。比如，jquery的跨浏览器问题。
3. 简化开发，能够让我们把主要的精力放在业务方面。
